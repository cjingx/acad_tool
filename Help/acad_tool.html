BGS Acad Tool


Indexing Tool

RINDEX

  • Stands for Rapid Indexing
  • It enables you to create Area_Function_Index for a row of conveyors using a single click.

NINDEX

  • Stand for Next Indexing
  • It will ask the user to select the conveyors one by one to update Area Function Indexing
Both RINDEX and PINDEX will update the AREA_FUNCTION, AREA_FUNCTION_INDEX and REF_DESC. Default prefix will be added to besd conveyors. RINDEX and NINDEX can be triggered interactively at the end of the command, so you can continue labelling when encouter customized block.

PINDEX

  • Stand for Prefix Indexing
  • It will ask the user for a prefix and update the conveyors' attributes.
  • It is usually useful for customized special conveyors

INDEXCONFIG

  • Opening the configuration file for the index tool.
  • Prefix for standard BESD equipment can be changed or omitted in this file.
  • Once the file is saved. we need to run LOADINDEXCONFIG to apply the changes.

LOADINDEXCONFIG

  • Apply settings from the configuration file.

Attribute Editor

ATE

  • Stands for Attribute Editor
  • It provides simple interface to edit attributes for single conveyor

ATT

  • Stands for Attribute Editor
  • It provides bulk editing option for conveyors.
  • It also allow user to swap the values of two attributes.

ATTX

  • Stands for Attribute Editor
  • It can add/remove attributes for all blocks definitions in the drawings

ACC

  • stands for Attribute Copy
  • It copy all attributes from one conveyor to another conveyor.

ATTCOLOR

  • stands for Attribute Coloring
  • It color all blocks in the drawing based on attribute value, so same attribute value will have same color
  • It also privide functionality to select a csv file which contains a list of attvalue, colorCode

Geometry Creation - Conveyor

CPSP

  • Stands for Crisplant Conveyor Support
  • It creates conveyor supports for multiple conveyors using one click.

CPRP

  • Stands for Crisplant Replace Support
  • It replace conveyor supports with new parameters.

CPFP

  • This feature is under devdelopment
  • Stands for Crisplant Filling Plate
  • It creates the 3d filling plate between conveyors.

BCV

  • It stands for Bottom Cover Generator
  • It helps you generate bottom covers by selecting corresponding conveyors
  • It is applicable to the following entity :
    • Straight CrisBelt
    • Nose Over CrisBelt

    Geometry Creation - Safety Fencing

    CPFEN

    • Stands for Crisplant Fencing Tool
    • It creates safety fencings.

    CPDOOR

    • Stands for Crisplant Fencing Door
    • It adds the door utilities to the fencing panel.

    FE

    • Stands for Fencing Edit
    • Used to quickly modify exiting fences
    • use Hxxxx and Wxxxx to change the height and width of the existing fencing

    UPDATEFENCETEXT

    • Stands for Update Fence Text
    • It will make the text of the fence block larger. so it is more friendly for printing.

    Geometry Creation - Steelwork

    STA

    • Stands for Stair
    • It creates steelwork stairs in a fast manner.

    RAIL

    • Stands for Handrail
    • It creates handrail on the platform boundary.

    FPP

    • Stands for Foot Plate
    • It creates footplate on the platform boundary.

    BRACE

    • It creates steel bracing for the platforms.

    STAINCA

    • It stands for Stair Inclination mark Add
    • It will add inclination mark for all the stair case which is created by bgs conveyor tool or bdk stair tool

    Geometry Creation - Access

    MVOLUME

    • It stands for Maintainence Volume Generator
    • It helps you generate maintainence volume for a number of conveyors in one go.
    • It is applicable to the following entity :
      • Straight CrisBelt
      • Nose Over CrisBelt
      • Power Curves Generated by BESD tool.
    Mvolume comes with two modes. solid mode and block mode. block mode will created unamed block with ref_desc of the conveyor.

    ROUTE

    • It stands for Route Creator
    • It convert a series of plyline into maintainence route symbols (undirectional)
    It will not calculate the distance.

    CHECKLENGTH

    • Calcuate the distance of a access route by clicking a series of points.
    • It will add the text label at the begining of the route.

    BESD Info Handling

    TESTCURVE

    • It stands Test Power Curve Radius
    • you will be prompted to input a radius.
    • the program will highlight any power curve which has a different radius.

    INCA

    • It stands for Inclination Addition
    • It adds inclination marks for besd conveyors.

    INCD

    • It stands for Inclination Delete
    • It remove all teh inclination marks for besd conveyors.

    CPX

    • It calculate the position of conveyors and update its attributes in conveyors.
    • These info includes start/end points coordinats, inclination angle etc.

    MAKEBESD

    • It stands for Make Besd Block
    • It adds essential besd attributes to customized blocks.
    BESD convention in block definition needs to be followed. insertion point will be cp4 point and block needs to be defined from left to right following material flow direction.

    Block Handling

    MX , MY , MZ

    • Move function in x, y , z directions.

    AX , AY

    • Move element in x or y direction so it match the orientation of another element.

    RB

    • It stands for Replace Block
    • It replace one block with another block.
    In the command of RB, attributes, layers are also updated.

    FC

    • It stands for Fast Copy
    • It copy the selected object based on WCS point (0, 0, 0)

    FP

    • It stands for Fast Paste
    • It paste the objects in clipboard based on WCS point (0,0,0)

    XREF Handling

    REFC

    • It stands for XREF Color
    • It changes the all layers of the attached xref to the color code.
    • This is extemely useful when doing layout, when you need to change multiple xrefs, and redraw them in one go.

    REFR

    • It stands for XREF RELOAD
    • It will check all attached XREF, if there is a new version, it will reload it.
    • This saves us from going to the xref panel and reload each drawing one by one.

    Searching Function

    FB

    • It stands for Find Block
    • It provides quick way to find blocks.

    FD

    • It stands for Finalize Drawing
    • It purges the block table, change layer to 0, and zoom extents in the x-y plane.

    FMB

    • It stands for Find Mirrored Block
    • It highlight mirrored block in the drawing.

    Measuring Function

    CPLEN

    • Stands for Crisplant Length Tool
    • It enable user to select the multiple BESD elements and calculate the overall length and travelling time
    If speed is not filled in the element, or default besd center line not defined in the block, the value will not be included.

    DIF

    • Measure the insertion point differences between two conveyors.

    VA

    • It stands for Veritcal Angle
    • It gives a quick calculation about the vertical angle of the conveyor.

    Layer Handling Functions

    LAYF

    • stands for layer freeze
    • freeze layer quickly using a command line regular expression (wildcards)
    • you can also save these wildcards in a file, so you could reuse them again and again

    LAYC

    • stands for layer coloring
    • color layer quickly using a command line regular expression (wildcards)
    • you can also save these wildcards in a file, so you could reuse them again and again

    BOM Generation Functions

    CPEXTRACT

    • Extract block info of the drawing to a database.
    • A project needs to be created to hold the block info of multiple drawings.

    CPGENERATE

    • Generate BOM list for a project (consists of mutiple drawings).
    • Drawing data needs to be extracted to the project first.
    • Attribute which needs to exported needs to be defined first.
